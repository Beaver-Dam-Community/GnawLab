## Fit notes

- Running shorts run slightly small. Customers between sizes are advised to
  size up.
- Training jackets run true to size with a relaxed fit.
- Compression tops run small. Most customers prefer one size up.

For exchanges driven by sizing, follow the standard exchange flow in the
Exchange Policy document.
