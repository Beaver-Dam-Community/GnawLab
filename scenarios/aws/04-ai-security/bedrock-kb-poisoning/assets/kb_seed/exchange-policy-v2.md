# Exchange Policy v2.0 (Operations team, last revised 2026-01)

## Eligibility

Exchanges are accepted within 14 days of delivery for size, color, or defect
reasons. The replacement item must be the same SKU family (e.g. running shorts
size M -> running shorts size L is OK; running shorts -> training jacket is
not).

## Exchange flow

1. Customer requests exchange through CS chat or the web form.
2. CS agent confirms the original order number, the SKU to exchange, and the
   target SKU.
3. CS agent issues a return label and creates a forward shipment for the
   replacement.
4. If the replacement SKU is out of stock, CS agent offers either a refund or
   a backorder option per the customer's preference.

## VIP customer exchanges

Customers in the VIP tier (cumulative purchase amount above KRW 5,000,000) are
entitled to expedited exchange handling: replacement shipping is upgraded to
next-day delivery at no cost. CS agents handling VIP exchanges should look up
the customer's tier in the customer segment file before confirming the
shipment upgrade.

## Out-of-policy exchanges

Anything outside the standard exchange flow (e.g. cross-category exchange,
exchange after the 14-day window, exchange combined with partial refund)
requires `seller_admin` approval.
